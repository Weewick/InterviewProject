# InterviewProject

## How to run
<p></p>

### Backend

<p></p>
<p>1. cd backend/</p>
<p>2. npm install</p>
<p>3. npm run build</p>
<p>4. npm run start</p>
<br>

### Frontend

<p></p>
<p>1. open new terminal</p>
<p>2. cd frontend/</p>
<p>3. npm install</p>
<p>4. npm run start</p>
<p>5. open http://localhost:4200/</p>
<br>
